# OpenClaude-MCP Integration

DeepFang can automatically screen every shell command generated by openclaude-mcp through the sanitizer before local execution. When enabled, openclaude-mcp becomes a gated tool: unsafe commands are blocked before they reach your shell.

## How It Works

```
openclaude-mcp generates a shell command
    → POST deepfang /api/sanitize  (regex threat scoring, <5ms)
        → if threat_score > 0.3:
        → POST deepfang /api/adjudicate  (DeepSeek LLM, ~2s)
        → if verdict != "approve": BLOCK — return error to Claude
    → Command is safe — execute locally
```

Without DeepFang, all commands generated by openclaude-mcp run directly on the host. With the guard, every command passes through the same pipeline that protects the air-gapped worker.

## Enabling the Guard

Add this to openclaude-mcp's `.env`:

```env
DEEPFANG_GUARD_URL=http://localhost:10956
```

When unset, openclaude-mcp runs as before — no breaking change.

## Guard Implementation

Add this wrapper in `openclaude-mcp/src/openclaude_mcp/tools/`:

```python
"""
DeepFang guard for shell commands generated by openclaude-mcp.

Enabled when DEEPFANG_GUARD_URL env var is set.
Calls deepfang_sanitize → deepfang_adjudicate before executing any command.
"""
import hashlib
import os

import httpx

DEEPFANG_URL = os.getenv("DEEPFANG_GUARD_URL", "")

_client: httpx.AsyncClient | None = None


def get_client() -> httpx.AsyncClient:
    global _client
    if _client is None:
        _client = httpx.AsyncClient(timeout=60.0)
    return _client


async def guard_command(cmd: str) -> dict:
    """Screen a command through DeepFang before execution.

    Returns: {allowed: bool, verdict: str, reason: str, threat_score: float}
    """
    if not DEEPFANG_URL:
        return {"allowed": True, "verdict": "bypass", "reason": "Guard not configured", "threat_score": 0.0}

    try:
        # Stage 1: Fast regex sanitizer
        sanitize_resp = await get_client().post(
            f"{DEEPFANG_URL}/api/sanitize",
            json={"content": cmd, "source": "openclaude-guard"},
            timeout=15.0,
        )
        sanitize = sanitize_resp.json()

        if not sanitize.get("allowed", False):
            return {
                "allowed": False,
                "verdict": "deny",
                "reason": sanitize.get("reason", "Sanitizer blocked"),
                "threat_score": sanitize.get("threat_score", 1.0),
            }

        threat_score = sanitize.get("threat_score", 0)
        if threat_score <= 0.3:
            return {
                "allowed": True,
                "verdict": "approve",
                "reason": f"Sanitizer passed (score {threat_score:.2f})",
                "threat_score": threat_score,
            }

        # Stage 2: LLM adjudication for ambiguous content
        adjudicate_resp = await get_client().post(
            f"{DEEPFANG_URL}/api/adjudicate",
            json={"content": cmd, "sanitize_result": sanitize},
            timeout=60.0,
        )
        adjudicate = adjudicate_resp.json()

        verdict = adjudicate.get("verdict", "deny")
        return {
            "allowed": verdict == "approve",
            "verdict": verdict,
            "reason": adjudicate.get("rationale", ""),
            "threat_score": threat_score,
        }

    except Exception as e:
        # Never block due to guard failure — we're a helper, not a critical path
        return {
            "allowed": True,
            "verdict": "guarded_allow",
            "reason": f"DeepFang unreachable: {e}",
            "threat_score": 0.0,
        }
```

Then in the main command execution path:

```python
from .guard import guard_command

async def execute_command(cmd: str) -> dict:
    # Screen through DeepFang if configured
    guard = await guard_command(cmd)
    if not guard["allowed"]:
        return {
            "success": False,
            "blocked": True,
            "reason": guard["reason"],
            "threat_score": guard["threat_score"],
        }

    # Safe to execute
    return await local_execute(cmd)
```

## Bypassing the Guard

To intentionally skip the guard for a specific command (e.g. a known-safe operation):

```python
from .guard import guard_command

async def execute_command(cmd: str, skip_guard: bool = False) -> dict:
    if not skip_guard:
        guard = await guard_command(cmd)
        if not guard["allowed"]:
            ...
    return await local_execute(cmd)
```

## Failure Mode

The guard **never blocks execution when DeepFang is unreachable**. If the sanitizer or adjudicator returns an error (`httpx.ConnectError`, timeout, etc.), the guard allows the command through with a warning logged. This ensures that a crashed DeepFang stack does not prevent openclaude-mcp from working.

The guard is a **safety net**, not a hard gate. Use DeepFang's full pipeline (`deepfang_pipeline`) for hard-gated execution in the air-gapped worker.
